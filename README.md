# 🚀 TasKify (FastAPI + React)
- TasKify is a simple and efficient ToDo List Application built with FastAPI (Backend) and React (Frontend).
- It helps you organize, track, and manage your daily tasks through a clean and user-friendly interface.

### 📌 Features
✅ Create new tasks
✅ View your list of tasks
✅ Update existing tasks
✅ Delete tasks
✅ Attractive UI using Bootstrap
✅ FastAPI-powered REST API
✅ Clean and simple project structure (No submodules)

### 🛠️ Tech Stack
#### 🖥️ Frontend
- React

- Bootstrap

#### 🛠️ Backend
- FastAPI

- SQLAlchemy

- Pydantic
